# Trendchaser 2026-07-07 Update — PRISMA-Codex Migration and Source Expansion

**TL;DR — the PRISMA loop moved from Claude Code Routines-style managed execution to GitHub Actions + Codex auth, and the documented source surface expanded from the old 67-enabled-channel loop to 101 enabled channels. The collector now watches more academic, conference, paper-index, implementation, and opaque-page sources while keeping the selection gate conservative.**

This release documents the PRISMA-Codex transition. Trendchaser remains a public technical report, not the private implementation repository. Tokens, private X account inventory, follow-list contents, and secret endpoints are intentionally excluded.

## Policy

The expansion is deliberately asymmetric: weak sources may discover candidates, but final selection still requires recency, source integrity, original URL checks, and either strong authority or cross-source confirmation. Cross-source metadata (`cross_confirm_count`, `cross_confirm_sources`, `cross_confirm_key`) improves ranking only; it does not override hallucination checks or freshness gates.
